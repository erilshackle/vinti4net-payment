💳 Vinti4Pay PHP SDK

Bem-vindo à documentação oficial do Vinti4Pay PHP SDK, uma biblioteca moderna e orientada a objetos para integração com o gateway Vinti4Net (SISP), permitindo pagamentos online, recargas, estornos e consultas de forma simples e segura.

⚙️ Recursos principais

  • 💰 Pagamentos por cartão Vinti4
  • 🔁 Estornos (Refund/Reversal)
  • 🧾 Geração de recibos HTML
  • 🧩 Verificação automática de fingerprint
  • 📦 Baseado em PHP 8+
  • 🧠 Código limpo e documentado

🚀 Comece agora

  1. Instale o pacote com o Composer.
  2. Configure o seu POS ID e POS Auth Code.
  3. Crie o formulário de pagamento.
  4. Receba e valide o retorno (callback).
